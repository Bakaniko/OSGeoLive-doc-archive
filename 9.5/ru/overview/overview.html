Геопространственные инструменты¶

Специализированные инструменты:

  • GMT - [Введение] - Картография и работа с геоданными
  • IPython Notebook - [QuickStart] - Интерактивный анализ разнородных данных и их визуализация в Python
  • OTB - [Введение] - Обработка изображений
  • R для пространственных данных - [Введение] - Математический анализ, статистика и программирование
  • Mapnik - [Введение] - Картографический рендеринг
  • TileMill - [QuickStart] - Стилизация и публикация карт
  • MapSlicer - [Введение] - Создание картографических тайлов
  • OSSIM - [Введение] - Обработка изображений
  • GeoKettle - [Введение] - Инструмент ETL (Extract, Transform and Load)

Специализированные инструменты для различных отраслей:

  • osgEarth - [Введение] - 3D-визуализация местности
  • MB-System - [Введение] - Картографирование морского дна
  • zyGrib - [Введение] - Карты прогнозов погоды

Другое геопространственное ПО (отсутствует на диске)¶

Доступное только для Microsoft Windows:

  • MapWindow GIS Lite - [Введение] - ГИС для OC Windows

Доступное в прошлых выпусках OSGeo-Live:

  • MapGuide Open Source - [Введение] - Веб-сервис, не влючается в состав OSGeo-Live после версии 5.0 (для уменьшения объёма диска)
  • Geopublisher - Геокаталог, исключён после версии OSGeo-Live 6.0
  • AtlasStyler - Редактор стилей, исключён после версии OSGeo-Live 6.0
  • MapFish - [QuickStart] - Фреймворк для веб-картографирования, исключён после версии OSGeo-Live 7.9.
