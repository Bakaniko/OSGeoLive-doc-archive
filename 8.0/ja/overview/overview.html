データ¶

空間データセット:

  • Natural Earth - 全球データセット
  • OSGeo North Carolina, USA Educational dataset
  • OpenStreetMap - OpenStreetMap からのサンプル抽出
  • NetCDF Data Set - 時系列の気温と降水量

地理空間ライブラリ¶

  • GDAL/OGR - [クイックスタート] - 地理空間データ変換ツール
  • JTS トポロジスイート (JTS) - Java トポロジスイート
  • GeoTools - Java GIS ツールキット
  • GEOS - C/C++ 空間ライブラリ
  • MetaCRS - 座標参照系の変換
  • libLAS - LiDAR データへのアクセス
  • Iris - [クイックスタート] - 気象学と気候学
